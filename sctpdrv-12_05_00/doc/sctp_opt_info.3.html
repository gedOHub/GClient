SCTP_OPT_INFO

NAME

sctp_opt_info - get SCTP socket information

LIBRARY

sctpsp.dll (sctpsp.lib)

SYNOPSIS

#include <winsock2.h>
#include <ws2tcpip.h>
#include <ws2sctp.h>

int
sctp_opt_info(int s, sctp_assoc_t asocid, int opt, void *arg, socklen_t *size);

DESCRIPTION

The sctp_opt_info() call provides a multi-os compatible method for getting specific getsockopt() data where an association identification needs to be passed into the operating system.
  • SCTP_RTOINFO
  • SCTP_ASSOCINFO
  • SCTP_PRIMARY_ADDR
  • SCTP_SET_PEER_PRIMARY_ADDR
  • SCTP_STATUS
  • SCTP_GET_PEER_ADDR_INFO
  • SCTP_AUTH_ACTIVE_KEY
  • SCTP_PEER_AUTH_CHUNKS
  • SCTP_LOCAL_AUTH_CHUNKS

ERRORS

The sctp_opt_info() function can return the following errors:
[WSAEINVAL]
The argument arg value was invalid.
[WSAEOPNOTSUPP]
The argument opt was not one of the above listed SCTP socket options.
[WSAEBADF]
The argument s is not a valid descriptor.
[WSAENOTSOCK]
The argument s is not a socket.
