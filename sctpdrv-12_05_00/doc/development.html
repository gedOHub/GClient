Developing SCTP applications using SctpDrv

The SctpDrv installer adds libraries and headers during a standard installation which can be used to develop SCTP applications.

SctpDrv installs its files to C:\Program Files\SctpDrv: an import library can be found in lib\sctpsp.lib

The DLL which applications which use SCTP specific functionality (as opposed to using nothing but the standard Winsock/BSD socket calls) need to link against is sctpsp.dll and can be found in C:\Program Files\SctpDrv\bin; on 64-bit systems, a 32-bit version of sctpsp.dll is installed to C:\Program Files (x86)\SctpDrv\bin .

To use the SCPI API or the SCTP specific structures, include ws2sctp.h from C:\Program Files\SctpDrv\inc\ws2sctp.h .
